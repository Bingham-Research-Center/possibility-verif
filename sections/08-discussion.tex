\section{Discussion and Future Work}\label{sec:discussion}

This paper has developed a self-contained verification framework for
possibilistic forecasts.  The core contribution is a five-number
scorecard (Section~\ref{sec:native_scorecard}) that evaluates a
subnormal possibility distribution on its own terms---without first
converting it to a probability.  Supporting this scorecard are a
possibility-to-probability conversion
(Section~\ref{sec:poss_prob_bridge}) that preserves the ignorance
signal as an explicit outcome rather than erasing it through
normalisation, and three complementary verification lanes
(Section~\ref{sec:three_lanes}) that together cover
categorical, probabilistic, and native possibilistic evaluation.
Worked examples in Section~\ref{sec:worked_examples} demonstrated
that each uncertainty component---possibility, ignorance, and
conditional necessity---independently contributes verification value,
so that collapsing the distribution into a single point forecast
discards actionable information.  Although the SPC convective outlook
categories served as the running example throughout, the framework
applies to any finite universe of discourse $\Omega$ over which a
subnormal possibility distribution is defined.

Although SPC convective outlook categories served as the running
example, the framework is domain-agnostic: any system producing
subnormal possibility distributions over a finite $\Omega$ can be
evaluated without modification.  Extension to continuous forecast
variables is straightforward via $\alpha$-cut discretisation.

\subsection{Generating Possibilistic Forecasts}%
\label{sec:generation}

A natural question arises: where do possibilistic forecasts come from?
This paper addresses verification, not generation, but several pathways
exist for producing the subnormal distributions the framework evaluates.

\textbf{Fuzzy inference systems.}  The most direct route uses fuzzy
rules whose membership functions are possibility distributions by
construction \citep{Klir1995-va}.  A fuzzy inference system maps input
meteorological variables through rule sets with fuzzy antecedents and
consequents; the aggregated output is naturally subnormal when rules
fire at partial strength, encoding the system's incomplete confidence
without normalisation.

\textbf{Ensemble-derived possibility.}  For each category, compute the
fraction of ensemble members predicting it---but do \textit{not}
normalise to $\max = 1$.  Let the ensemble spread speak: if members are
scattered across categories, the maximum fraction is low and $\ign$ is
high; if members cluster, the maximum is high and $\ign$ is low.  The
raw ensemble ``support'' for each outcome is already a possibility-like
quantity.  This is the most accessible pathway for a NWP audience:
ensemble spread, formalised as subnormality.

\textbf{Severity--confidence matrices.}  Operational warning frameworks
already encode possibilistic structure.  The UK Met Office National
Severe Weather Warning Service (NSWWS) uses a two-dimensional matrix
of likelihood and impact to determine warning levels
\citep{Neal2014-MISSING}.  The impact axis encodes what \textit{might}
happen (the shape of the possibility distribution), while the likelihood
axis encodes how confident the forecaster is (akin to subnormality).
Figure~\ref{fig:severity_matrix} adapts this concept to the paper's SPC
framework, mapping severity categories against confidence levels to show
how the joint signal varies from ``monitor'' (low severity, high
ignorance) to ``act now'' (high severity, low ignorance).  The
verification framework developed here formalises this operational
intuition with precise metrics.

To connect the matrix to the verification framework, consider a cell at
\spc{MDT} severity with high confidence ($\ign = 0.10$).  The implied
distribution might be
$\pi = (0.00,\, 0.00,\, 0.05,\, 0.15,\, 0.90,\, 0.10)$---exactly
Scenario~A from Section~\ref{sec:worked_examples}.  The five-number
scorecard then evaluates whether that matrix cell's implied distribution
is well-calibrated: $\alpha^* = 1.0$ and $\delta = +0.778$ when
\spc{MDT} verifies, but $\alpha^* = 0$ and $\delta < 0$ when it does
not.  Aggregating scorecard values across all days that fall in a given
matrix cell reveals whether the operational severity--confidence mapping
is empirically justified.

\subsection{Limitations}\label{sec:limitations}

\begin{itemize}[leftmargin=2em]
\item \textbf{Sample-size requirements} for the component-value
    assessments (Section~\ref{sec:component_value}) have not been
    established.  At pilot scales (${\sim}90$ station-days), the
    stratified assessments (ignorance and necessity) will have limited
    statistical power; at least 30 cases per stratum are needed for
    reasonable precision, and illustrative case studies should supplement
    formal tests.
\item \textbf{No possibilistic skill score} is defined, because
    constructing a principled possibilistic climatology---the baseline
    needed for skill---remains an open problem.
\item \textbf{Closed-world assumption:} the probability conversion assumes the true
    category is always in $\Omega$.  For open-world applications the
    framework would need modification to allow the ignorance outcome to
    verify as ``correct.''
\end{itemize}

\subsection{Future Directions}\label{sec:future_directions}

\begin{itemize}[leftmargin=2em]
\item \textbf{Possibilistic baselines and credal-set bounds:} formal
    exceedance bounds grounded in the possibility--necessity duality,
    rigorously derived for subnormal distributions.
\item \textbf{Multivariate verification:} joint possibility
    distributions over correlated hazards (e.g., simultaneous wind and
    hail) with an extended scorecard for multidimensional $\Omega$.
\item \textbf{Extension to continuous variables:} $\alpha$-cut
    discretisation and spatial verification maps.
\item \textbf{Conformal prediction:} formal comparison of conformal
    coverage guarantees with possibilistic intervals.
\item \textbf{Operational dashboards:} real-time scorecard monitoring
    across successive forecast cycles.
\item \textbf{LLM-mediated communication:} translation of the
    possibilistic triplet ($\poss$, $\ign$, $\Nc$) into tiered
    plain-language advisories (Lawson, in prep.).
\item \textbf{Model-development trajectories:} extending
    Figure~\ref{fig:performance_diagram} to track successive model
    versions as connected points.
\end{itemize}
