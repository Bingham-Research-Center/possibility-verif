\subsection{Structured Uncertainty and LLM-Mediated Communication}%
\label{sec:llm_communication}

The three-component representation ($\poss$, $\ign$, $\Nc$) was designed
not only for verification but also for downstream communication.  In a
companion system (Lawson, in prep.), a large language model (LLM)
translates the structured triplet into plain-language advisories via a
four-stage pipeline:

\begin{enumerate}[leftmargin=2em]
\item \textbf{Monte Carlo clustering.}  Ensemble members (here, GEFS)
    are clustered into representative scenarios, each tagged with
    possibility values and an ignorance estimate.
\item \textbf{Structured extraction.}  The possibilistic triplet
    ($\poss$, $\ign$, $\Nc$) and supporting meteorological context are
    serialised as JSON, providing the LLM with machine-readable
    uncertainty metadata rather than raw gridded fields.
\item \textbf{Guarded prompt construction.}  Pre-set guardrails enforce
    citation of source data, caveats when $\ign$ exceeds a threshold
    (``conditions are unusual; confidence is limited''), and
    prohibition of fabricated precision.  Custom messages---e.g.,
    regulatory triggers, operational deadlines---are injected per
    stakeholder tier.
\item \textbf{Tiered advisory generation.}  The LLM produces advisories
    at three complexity levels (public, stakeholder, technical) rendered
    as automated PDFs (Figure~\ref{fig:ffion_advisory}).
\end{enumerate}

\begin{figure*}[t]
\centering
\includegraphics[width=0.80\textwidth]{fig9_ffion_advisory.pdf}
\caption{Schematic of LLM-mediated risk communication from a
    possibilistic forecast.  \textbf{Left:} structured JSON input
    containing the three-component triplet and full distribution.
    \textbf{Right:} stakeholder-level advisory generated by an LLM
    with pre-set guardrails.  The ignorance signal ($\ign = 0.25$)
    produces hedged language; conditional necessity ($\Nc = 0.73$)
    produces confident language---both grounded in the forecast data
    rather than manufactured by the model.  Adapted from Lawson
    (in prep.).}
\label{fig:ffion_advisory}
\end{figure*}

Run-to-run consistency is maintained by pinning absolute paths to both
input NWP data and outlook files; the embedded AI agent (Anthropic's
Claude, operating as a CLI tool) requires approximately three minutes of
inference time per outlook cycle, ingesting data slices and forecast
context before generating each advisory document.  The explicit ignorance
signal is critical: it gives the LLM a principled basis for expressing
uncertainty rather than manufacturing false precision from a normalised
distribution.  Conditional necessity provides the complementary trigger
for confident statements.

The verification framework developed here closes the loop: the
five-number scorecard evaluates whether the structured output that feeds
communication is itself trustworthy.  Full details of the LLM
translation layer are beyond the present scope; the point relevant to
verification is that possibilistic forecasts produce machine-readable
uncertainty metadata that probabilistic point forecasts do not.
